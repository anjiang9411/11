# AI互动面板功能使用指南

## 功能介绍

当你和AI角色聊天时，可以要求AI展示他们的浏览器记录、聊天列表、聊天截图、应用列表等信息。AI会生成可互动的HTML面板，你可以点击查看详细内容。

## 支持的面板类型

### 1. 浏览器历史记录 (browser-history)
- 展示AI角色最近访问的网页
- 包含网页标题、URL、访问时间
- 点击可查看详细信息

### 2. 聊天列表 (chat-list)
- 展示AI角色的聊天对象列表
- 显示头像、昵称、最后一条消息、时间、未读数
- 点击可查看更多信息

### 3. 聊天截图 (chat-screenshot)
- 展示AI角色与其他人的聊天记录
- 以缩略图形式展示
- 点击可查看完整聊天内容

### 4. 应用列表 (app-list)
- 展示AI角色最近使用的应用
- 显示应用图标、名称、最后使用时间
- 点击可查看详情

## 如何触发

只需要在聊天中向AI提问，例如：

- "给我看看你的浏览记录"
- "你最近在看什么网页？"
- "你在和谁聊天？"
- "给我看看你的聊天列表"
- "你们聊了什么？给我看看截图"
- "你最近用什么app？"
- "你手机里装了什么应用？"

## AI会如何回应

AI会根据：
1. **人设性格**：外向的角色更愿意分享，内向的可能会犹豫
2. **好感度**：好感度高的时候更愿意展示私密信息
3. **当前情绪**：心情好的时候更容易答应

AI可能的回应：
- ✅ 直接发送面板："好吧，给你看看[面板]"
- ⚠️ 有条件同意："就给你看一眼哦[面板]别告诉别人！"
- ❌ 拒绝："这个不能给你看啦"
- 🤔  反问："为什么突然想看这个？"

## 面板交互

### 列表视图
- 点击面板卡片，打开列表弹窗
- 显示所有项目
- 支持滚动查看

### 详情视图
- 点击列表中的项目，查看详细信息
- 可以返回列表
- 可以直接关闭

## 示例对话

### 示例1：查看浏览记录
```
用户：你最近在看什么网页啊？
AI：嗯...好吧，给你看看我最近的浏览记录
    [显示可点击的面板卡片]
    不许笑我哦！

[用户点击面板]
→ 显示浏览记录列表：
  • 如何做红烧肉 - 14:30
  • 北京旅游攻略 - 10:15
  • Python教程 - 昨天

[用户点击某条记录]
→ 显示详细信息和URL
```

### 示例2：查看聊天列表
```
用户：你在和谁聊天呀？
AI：哎呀，被你发现了
    [显示聊天列表面板]
    就这几个人啦，你可别吃醋~

[用户点击面板]
→ 显示聊天列表：
  • 张三 - "晚上一起吃饭吗？" - 18:30 [2]
  • 李四 - "明天见~" - 昨天
```

### 示例3：好感度低时拒绝
```
用户：给我看看你的聊天记录
AI：这个不能随便给你看吧
    我们又不是很熟...
```

## 技术说明

### 数据格式
AI会生成JSON格式的结构化数据，系统自动解析并渲染为交互界面。

### 隐私保护
- AI会根据人设和好感度决定是否展示
- 可以拒绝或只展示部分信息
- 符合角色的真实人类行为

## 注意事项

1. **真实性**：AI展示的数据会符合角色的人设和生活状态
2. **互动性**：不是简单的文本，而是可点击查看的交互界面
3. **情感化**：AI会根据情绪状态决定如何回应
4. **合理性**：浏览记录、聊天对象等都会与角色背景相符
